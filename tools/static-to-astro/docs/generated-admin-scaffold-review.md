# Generated Admin Scaffold Review

**Phase:** G-5w-d — sandbox scaffold review (read-only)  
**Status:** review only · no runtime connection · no `src/pages/admin` changes

---

## 1. Purpose

G-5w-d reviews the sandbox admin scaffold generated by [G-5w-c sandbox apply](./admin-scaffold-writer-sandbox-apply.md) before proceeding to **G-5x staging runtime shell integration**.

The review confirms:

- Expected file structure
- Page safety markers (noindex, scaffold-only, no runtime)
- Config consistency (sections, permissions, storage, publish policy)
- Manifest consistency (generated-files-manifest, rollback-manifest, writer-report)
- `readyForG5x` gate for the next phase

## 3. Prerequisites

Generate G-5s package and G-5w-c sandbox (local only, not committed):

```bash
node tools/static-to-astro/scripts/generate-admin-scaffold-dry-run.mjs \
  --site-config tools/static-to-astro/config/sites/gosaki.site-config.example.json \
  --out-dir tools/static-to-astro/output/admin-scaffold-packages/gosaki

rm -rf tools/static-to-astro/output/admin-writer-sandbox/gosaki

node tools/static-to-astro/scripts/write-admin-scaffold.mjs \
  --package-dir tools/static-to-astro/output/admin-scaffold-packages/gosaki \
  --target-dir tools/static-to-astro/output/admin-writer-sandbox/gosaki \
  --apply \
  --approval-id G-5w-d-review-fixture
```

## 5. Checklist

### File structure

- [ ] `admin/README.md`
- [ ] `admin/pages/*.astro` (9 pages)
- [ ] `admin/config/*.json` (6 files)
- [ ] `admin/manifests/*` (3 files)
- [ ] No unexpected extra files

### Page safety (each page)

- [ ] `<meta name="robots" content="noindex,nofollow,noarchive" />`
- [ ] Scaffold only / generated sandbox only
- [ ] Not connected to runtime
- [ ] No Supabase Auth / DB / Storage / GitHub / FTP
- [ ] Production disabled
- [ ] No Supabase imports, fetch, or secret env references

### Config consistency

- [ ] Sections: dashboard, profile, schedule, discography, links, news, media, publish, settings
- [ ] Permissions: viewer, editor, admin
- [ ] Storage: discography_cover, schedule_home, schedule_flyer
- [ ] Publish: staging enabled, production disabled
- [ ] Preview plan: productionReady false, connectedToRuntime false

### Manifest consistency

- [ ] `generated-files-manifest.json`: mode apply-to-sandbox, overwrittenFiles []
- [ ] Safety flags: runtimeFilesWritten false, productionTouched false, srcPagesAdminTouched false
- [ ] `rollback-manifest.json`: delete-generated-sandbox-directory, safeToDeleteTargetDir true
- [ ] `writer-report.md`: apply-to-sandbox safety statements
