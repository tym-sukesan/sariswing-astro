# CMS Kit 管理画面のご説明 — Musician Basic

**対象:** ミュージシャン、音楽教室、小規模スタジオの方  
**用途:** デモ前後に渡す・読み上げる説明文

---

## はじめに

この資料は、**あなた専用のホームページを、ご自身でも更新しやすくする** ための管理画面のイメージをお伝えするものです。

今お見せしている画面は **完成版ではありません**。レイアウトや項目名を一緒に確認するための **試作（イメージ画面）** です。実際の保存や公開は、正式導入後に段階的に有効になります。

---

## CMS Kit とは

CMS Kit は、**ミュージシャンや小規模事業者向けに、必要な機能だけを選んで作る** Web サイトの仕組みです。

- **軽く・速く** — 不要な機能を省き、表示も管理もスムーズに
- **更新しやすく** — ライブ予定やお知らせを、ご自身で追加・編集できる
- **必要な機能に絞る** — プロフィール、スケジュール、ニュース、リンクなど、本当に使う項目だけ
- **安全に公開** — 本番に出す前に、確認用サイト（staging）で見た目をチェック

大規模な制作会社向けの複雑なシステムではなく、**日々の更新に必要なところに集中** した設計です。

## いまお使いのサービスから移行するメリット

Wix、Jimdo、Studio などのサービスをお使いの方にも向いています。

| 観点 | CMS Kit の考え方 |
| --- | --- |
| 月額コスト | 必要な機能に絞り、長期的に無駄の少ない運用を目指す |
| 表示速度 | 軽いページ構成で、訪問者にも読みやすい |
| 更新のしやすさ | 管理画面から、よく触る項目だけを分かりやすく配置 |
| 独自ドメイン | ご希望に応じて、お持ちのドメインで運用可能（導入時に相談） |
| サポート | 導入・更新方法を一緒に確認しながら進められる |

「今のサービスが悪い」というより、**より軽く・分かりやすく・本人が使いやすい形** を目指す、という考え方です。

## ご本人が更新できる範囲

正式導入後は、権限設定により異なりますが、一般的には次のような更新が想定されます。

| 項目 | 説明 |
| --- | --- |
| Profile | プロフィールや写真を更新できます |
| Schedule | ライブ・レッスン予定を追加できます |
| News | お知らせを投稿できます |
| Media | 画像を管理できます |
| Publish | 確認後に公開できます |

その他、Discography（作品情報）、Links（SNS・外部リンク）なども、テンプレートに応じて利用できます。

**制作側（管理者）が行うことが多い作業**

- 初回のサイト構成・デザイン調整
- 本番公開の最終承認
- ドメイン・サーバー設定
- 困ったときのサポート

---

## Staging と本番の違い

| 種類 | 説明 |
| --- | --- |
| **Staging（確認用）** | 本番と同じ見た目で確認できる、非公開のテスト環境。ここで誤字・画像・リンクをチェックします。 |
| **Production（本番）** | 一般の方が見る公式サイト。公開前に必ず確認用でチェックし、**お客様の承認後** に反映します。 |

「保存したらすぐ本番に出る」わけではありません。安心して試せるステップを用意します。

---

## 本番公開は承認制

- 本番への反映は、**管理者とお客様の双方の確認** を経て行います。
- 急ぎの修正も、まず確認用で見てから本番に出す流れを推奨します。
- 間違えて消してしまった場合の復旧方法も、導入時にご説明します。

---

## 画像の権利確認

- プロフィール写真、ライブ写真、ジャケット画像などは、**ご自身の素材** または **使用許可のある素材** をお使いください。
- インターネットから無断で持ってきた画像は使えません。
- 不明な場合は、導入前に一緒に確認します。

---

## 月額運用・保守について

正式な料金は案件ごとに異なりますが、一般的には次のような考え方です。

- **ホスティング・ドメイン・更新サポート** を含む月額または年額
- 大きな機能追加は別途見積もり
- 日常的なお知らせ・スケジュール更新は、ご自身で行えるようにサポート

「何かあったときに相談できる」安心感と、「自分で更新できる」自立感のバランスを大切にしています。
